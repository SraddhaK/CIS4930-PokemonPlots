# CIS4930-PokemonPlots
The Team: Sam Miller (SSM23A), Sofie Szlezak (SSS23J), Aiden Duncan (ALD24C), Sraddha Karthik (SK23BJ)
This Python code analyzes various Pokemon attributes over multiple generations to find discrepancies and comparisons in aspects like weight, attack power, and power creep over time. Using datasets gathered from multiple sources, we assess differences in the game of Pokemon as it shifts over time to promote user engagement.
Research Questions:
  - How do changes in Pokemon power levels across new iterations of the game affect power creep over time?
  - How do variations in Pokemon statistics such as type, weight, or power translate into different visual representations?
  - In changing Pokemon base statistics to increase user engagement, how does this change gameplay in terms of the statistics it is based around?
