# CIS4930-PokemonPlots
The Team: Sam Miller (SSM23A), Sofie Szlezak (SSS23J), Aiden Duncan (AJD24C), Sraddha Karthik (SK23BJ)
This Python code analyzes various Pokemon attributes over multiple generations to find discrepancies and comparisons in aspects like weight, attack power, and power creep over time. Using datasets gathered from multiple sources, we assess differences in the game of Pokemon as it shifts over time to promote user engagement.
Dataset: https://www.kaggle.com/datasets/mariotormo/complete-pokemon-dataset-updated-090420
Research Questions:
  - How do changes in Pokemon power levels across new iterations of the game affect power creep(where power/stats increase over time)?
  - Which Pokemon typing(fire, water, etc.) has the greatest power?
  - What is the most frequent typing and how do the base stats compare to other types?
